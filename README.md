# Meemo

Сервис-обертка над S3.

### Функционал
1) Регистрация-авторизация пользователей на уровне приложения, а не s3
2) Упрощенный интерфейс взаимодействия с s3
3) Шифрование файлов
4) Аудит приложения(логи, метрики)
5) Сжатие файлов

### Стэк:
1) Go 1.24, Echo 4.13.4
2) Ceph 19.2.3
3) Redis 8.2.0
4) Postgres 16.0
5) Kafka 4.0.0
